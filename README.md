## Project Overview
This personal portfolio website showcases the intersection of **Systems Logic** and **User Experience Design**. The project focuses on creating a high-performance, aesthetically modern interface that utilizes spatial design elements to communicate technical expertise.

## Key Features
* **Interactive 3D Environment:** Integrated via **Spline**, providing a dynamic spatial background that responds to user interaction.
* **Responsive Architecture:** Fully optimized for cross-device compatibility using a mobile-first approach.
* **AOS Integration:** Smooth "Animate On Scroll" transitions to improve content readability and user engagement.
* **Pre-loader Logic:** A custom "Initializing System" loader implemented to manage asset-heavy background loads, ensuring a seamless UX.
